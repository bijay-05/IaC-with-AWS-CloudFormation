# Configurations for various AWS Resources

## ECR

### Pull through cache

The Pull through cache configuration for your private registry displays the total number of pull through cache rules you have created in this Region. A pull through cache rule may be created to cache repositories from an upstream registry in your Amazon ECR private registry.

By default, when a pull through cache rule is created, any IAM principal that has permission to pull images from your private registry can use a pull through cache rule as well. You can use registry permissions to further scope down these permissions to specific repositories.

### Replication

The Replication configuration for your private registry displays the number of replication rules you have created.
